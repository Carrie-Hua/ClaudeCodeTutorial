## Notes:
1. Press Shift + Tab to switch different mode. e.g. plan mode: will not write code automatically
2. Include "Ask me questions to clarify the product requirements, technical requirements, engineering principles, and hard constraints" will get a richer discussion in the plan mode
3. Create Claude.md:
- use /init command to start a brand new Claude.md (not recommended)
- use the prompt in Commands and Prompt file 
4. To view a markdown file: Command + Shift + V
5. YOLO mode: allow you to run Claude with the skip permissions switch. Enable the mode through Shift + Tab
